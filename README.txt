Personal Website Pack — 2025-09-16

Files:
- index.html      → Your homepage
- headshot.jpg    → Square headshot used by index.html
- cv.pdf          → Linked in the navbar button (rename your CV to cv.pdf if you replace it)

Quick Deploy (GitHub Pages):
1) Create a repo named: <your-username>.github.io
2) Upload index.html, headshot.jpg, cv.pdf to the repo root (no subfolders).
3) In the repo: Settings → Pages → Deploy from branch.
4) Visit: https://<your-username>.github.io

Optional:
- Replace headshot.jpg with your own (keep the filename).
- Edit index.html text (name, interests, links) in any editor.
